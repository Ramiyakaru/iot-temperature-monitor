# IoT Temperature Monitoring System
An IoT project built using ESP8266, MQTT, Mosquitto, Node-RED, and WS2812 RGB LEDs.

## Features

- Read temperature using Keyestudio analog sensor
- Display readings in Arduino Serial Monitor
- Publish temperature data using MQTT
- Visualize data in Node-RED dashboard
- RGB LED color indication based on temperature thresholds

## Technologies Used

- ESP8266
- Arduino IDE
- MQTT
- Mosquitto
- Node-RED
- Adafruit NeoPixel Library

## Hardware

- ESP8266 NodeMCU
- Keyestudio Temperature Sensor
- WS2812 RGB LED Strip
